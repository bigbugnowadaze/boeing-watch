# Boeing Watch

An independent accountability dashboard for The Boeing Company.

Stage 1 (the public static site) is being developed on the
`claude/boeing-watch-stage-1-3Mpy4` branch. The pull request opened from
that branch is the review checkpoint before Stage 1 deploys to
`boeingwatch.org` via Cloudflare Pages.

The launch package — `EXPLAINER.md`, `BUDGET.md`, `HANDOFF.md`,
`README-launch.md`, and the six `agent-prompts/` — sits on this branch as
the reference baseline. Stage 2 (Workers, D1, agents, cross-platform
posting) follows once Stage 1 is verified live.
